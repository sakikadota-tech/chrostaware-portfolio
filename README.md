# ChroStaware ｜ AI開発ポートフォリオ

> 内部統制を、あなたの会社に。AIで、経営者の思想をスタイリッシュな"仕組み"に。

AI開発の成果物をまとめたポートフォリオサイトです。Claude Code で制作しました。

## 構成

- `index.html` — サイト本体（1ファイル完結 / HTML・CSS・JS）
- `work-control.jpg` / `work-efficiency.jpg` / `work-learning.jpg` — 実績画像

## 実績を追加・編集するには

`index.html` 内の `works = [ ... ]`（後半のスクリプト部分）を編集します。
1作品 = 1ブロック（画像・タイトル・課題・作ったもの・結果）。ここを足すだけでギャラリーにカードが増えます。

## デザインシステム

配色・雰囲気・コンポーネントの作法は「Quiet Future — Governance, made human」として
別途 YAML（`design-system.prompt.yaml`）に抽象化しています。
